# dummy-site
Dummy site made from `scratch` with core Node JS, native CSS &amp; Vanilla JS

# to run the app:
- in `api` folder, create .env file (see .env.default for reference)
- create local db

# run these scripts:
- npm run db-migrate
- npm run db-migrate-insert-init
- npm run dev

# on browser:
- http:localhost:3000
